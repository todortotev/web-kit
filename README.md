# web-kit

1. Install yarn
https://classic.yarnpkg.com/en/docs/install

2. Build libs

```
 cd packages/components
 yarn build
 cd ../widgets
 yarn build
```

3. Run storybook ( while inside the target folder - either widgets or components )

```
 yarn storybook
```
